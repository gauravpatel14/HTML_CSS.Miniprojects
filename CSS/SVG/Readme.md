# SVG Showcase

A single HTML file demonstrating core SVG shapes, animations, and gradients.

## What's Included

- **Shapes** – Circle, rectangle, line, polygon, and custom `<path>` elements
- **Text** – Inline SVG text rendering
- **Animations** – Moving circle, color-cycling rect, scaling circle, line draw effect
- **Gradients** – Linear and radial gradient fills
- **Heart** – Animated pulsing heart with gradient color shift

## Getting Started

```bash
git clone https://github.com/your-username/svg-showcase.git
cd svg-showcase
open index.html
```

No dependencies. No build step.
