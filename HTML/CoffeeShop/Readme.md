# ☕ Coffee Shop

A simple static coffee shop website built with HTML and CSS.

## Sections

- **Header** – Logo and navigation links
- **Hero** – Full-width coffee image with a Learn More button
- **About Us** – Shop description and specialities
- **Menu** – Coffee, Tea, and Pastries listed in a table
- **Footer** – Copyright notice

## Tech Stack

- HTML5
- CSS3 (Georgia serif, warm brown theme)

## Getting Started

```bash
git clone https://github.com/your-username/coffee-shop.git
cd coffee-shop
open home.html
```

No dependencies. No build step.
