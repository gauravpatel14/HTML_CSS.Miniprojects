# Amazon Landing Page Clone

A pixel-faithful Amazon India landing page built with pure **HTML & CSS** — no frameworks, no dependencies, no external images. All visuals are rendered using inline **SVG illustrations**.

---

## Preview

The page includes:
- **Navbar** — logo, location, search bar, account & cart links
- **Sub-navigation** — department links bar
- **Hero Section** — full-width SVG banner with deals headline
- **Category Cards** — Electronics, Fashion, Home & Kitchen, Books (all SVG)
- **Promotional Banners** — Electronics Deals & Amazon Prime (SVG)
- **Footer** — links and copyright

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Markup | HTML5 |
| Styling | CSS3 (Flexbox, Grid, Variables) |
| Graphics | Inline SVG (no external images) |
| Fonts | System fonts (`Amazon Ember`, `Helvetica Neue`) |

## Getting Started

No build tools or installation required.

```bash
# Clone or download the file
git clone https://github.com/your-username/amazon-landing.git

# Open directly in browser
open amazon.html
```

Or simply double-click `amazon.html` in your file manager.

## Features

- **Zero dependencies** — runs offline, no CDN or network needed
- **All SVG graphics** — hero, cards, and banners use handcrafted inline SVGs
- **Responsive-ready layout** — CSS Grid and Flexbox used throughout
- **Amazon color palette** — `#131921`, `#232f3e`, `#ff9900` faithfully matched
- **Hover states** — nav links, buttons, and cards have interactive hover styles

## Customization

All colours are set inline via CSS. To change the theme, find and replace:

| Variable | Default |
|----------|---------|
| Primary dark | `#131921` |
| Secondary dark | `#232f3e` |
| Accent orange | `#ff9900` |
| Link teal | `#007185` |
